2)Konfersji liczb

Wczytanie zmiennej z użyciem prompt wykonywane jest jako ciąg znaków dlatego konieczna jest
konwersji z ciągu znaków na wartości liczbowe aby wykonywac obliczenia.

3)Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:
  • Nazwa zmiennej musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery
    (bez polskich znaków), cyfry oraz "$" i "_".
  • Wielkość liter ma znaczenie.
  • Należy nie używać słów kluczowych JavaScript jako nazw zmiennych
4)typy zmiennych w Javascript
  • Number --> Typ liczbowy (całkowite albo zmiennoprzecinkowe)
  • String --> Typ tekstowy
  • Boolea --> Typ logiczny
  • Null --> Pusta zawartość
  • Undefined --> Typ niezidentyfikowany
  • Object --> Obiekt
